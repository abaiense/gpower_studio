## História e Empresa

### Linha do Tempo

**2015 — Fundação como Shedul**
William Zeqiri (ex-VP de Tecnologia da Dubai Holding) e Nick Miller identificaram que 52% dos salões nos EUA ainda agendavam com papel e caneta. Lançaram o Shedul com MVP cloud e modelo freemium radical: zero assinatura mensal. A estratégia foi remover barreiras financeiras para maximizar adoção rápida.

**2016 — Product-Market Fit**
Atingiram 1.000 estabelecimentos parceiros em meados de 2016, validando o modelo.

**2017 — Série A ($6M)**
Investimento liderado por Target Global (com BECO Capital e FJ Labs). Recursos usados para expansão nos EUA, Reino Unido e Austrália.

**2018 — $1 Bilhão em GMV cumulativo**
A plataforma atingiu $1 bilhão em valor bruto de mercadorias processado, com equipe enxuta.

**2019 — Rebranding para Fresha + Série B**
- Junho 2019: Shedul se torna **Fresha**, unificando o software e o marketplace sob uma marca global.
- Série B de $20 milhões liderada por Partech Partners.
- Abertura de escritórios em Nova York, Dubai e Varsóvia.
- Lançamento do **Fresha Plus**: primeiros recursos pagos (pagamentos integrados, automação de marketing).

**2021 — Série C ($152,5M)**
- Rodada inicial de $100M liderada por General Atlantic.
- Extendida para $152,5M com participação de BECO Capital, Partech, Target Global e FMZ Ventures.
- Valuation superando $640 milhões.
- Foco: expansão em fintech (pagamentos embarcados, Fresha Capital).

**2022 — Extensão Série C ($52,5M)**
Rodada adicional de $52,5M para aceleração global.

**2024 — Financiamento J.P. Morgan ($31M)**
Venture debt de $31 milhões do J.P. Morgan Asset Management (agosto 2024) para investimento em machine learning e robótica com IA.

**2025 — Mudanças no modelo de preços**
Introdução de assinaturas mensais (plano Solo e Team), encerrando o modelo completamente gratuito de software.

**2026 — Unicórnio ($80M KKR)**
- Maio 2026: Rodada de crescimento de $80M liderada pela KKR.
- Valuation oficial: $1 bilhão+.
- Crescimento: 60%+ ao ano.
- Receita run-rate: $140M+.
- Parceria com Adyen para Fresha Capital (empréstimos embutidos).
- Lançamento do AI Intelligent Scheduling.

## Modelo de Negócio e Receita

O Fresha opera como um **marketplace de dois lados** (negócios de beleza ↔ clientes consumidores) combinado com **software SaaS de gestão**. A monetização é multi-camada:

### Fontes de Receita

| Fonte | Detalhes |
|---|---|
| **Assinatura mensal** | $19,95/mês (solo) ou $14,95/membro/mês (team) |
| **Taxa de marketplace** | 20% sobre o valor do 1º agendamento de clientes novos via marketplace (mínimo $6) |
| **Processamento de pagamentos** | 2,19% + $0,20 por transação (presencial); taxas maiores para online/Amex |
| **SMS/WhatsApp pay-per-use** | Após cota gratuita: $0,35/SMS ou $0,10–$0,75/WhatsApp (valores EUA) |
| **Marketing email** | Após 50 grátis: $0,01/email |
| **Add-ons** | Loyalty, Insights, Team Connect, Google Rating Boost, Xero, Data Connector |
| **Hardware** | Terminal Fresha S1: $499 (venda única) |
| **Fresha Capital** | Financiamento a negócios (porcentagem retida nas vendas) |
| **Fresha Store** (não confirmado) | Comissão sobre vendas de produtos online |

### Estratégia do Modelo Freemium

O modelo original era 100% gratuito (sem assinatura), gerando receita apenas via marketplace e pagamentos. Em 2025, introduziram assinaturas mensais, mas mantiveram o marketplace como canal principal de crescimento e como diferencial competitivo versus concorrentes como Vagaro e Square.

## Como o Marketplace Funciona

### Para os Negócios

1. **Criação de perfil público** no marketplace Fresha com fotos, serviços, preços, equipe e avaliações.
2. **Verificação de perfil** ("Fresha Verified") melhora ranking e credibilidade.
3. **Visibilidade algorítmica** baseada em: localização geográfica, avaliações positivas, completude do perfil, serviços disponíveis online, turnos ativos cadastrados e políticas de pagamento ativas.
4. **Taxa de 20% apenas no 1º agendamento** de um cliente novo vindo do marketplace. Visitas repetidas: gratuitas.
5. **Mínimo de $6 por novo cliente** mesmo para serviços de baixo valor.

### Para os Clientes

1. Buscam serviços no app ou site Fresha.
2. Filtram por localização, tipo de serviço, preço, avaliações, disponibilidade.
3. Selecionam profissional e horário.
4. Criam conta Fresha (obrigatório).
5. Confirmam e pagam (online ou presencialmente).
6. Recebem lembretes automáticos.
7. Deixam avaliação após atendimento.

### Fatores de Ranking no Marketplace

- Proximidade ao cliente
- Volume e qualidade de avaliações
- Status "Fresha Verified"
- Serviços ativados para booking online
- Turnos de equipe atualizados
- Políticas de pagamento/cancelamento ativas
- Velocidade de resposta a avaliações

### Integrações de Canal

- **Reserve with Google**: agendamentos direto pelo Google Search e Maps
- **Facebook/Instagram**: botão "Agendar Agora" nas páginas de negócio
- **Link no website**: widget ou link para página de booking customizada
- Todas integradas em tempo real com o calendário Fresha

- Precificação dinâmica: preços menores em horários de baixa demanda, preços de última hora.
- Otimização contínua de disponibilidade, staffing e fluxo de atendimentos.

### Módulo 4: Pagamentos — Fresha Pay

- Taxa presencial: 2,19% + $0,20 por transação (USD)
- Taxas variam por região, método e bandeira (Amex mais caro)
- Pagamentos online para depósitos e reservas seguras

**Recursos:**
- Repasses diários (daily payouts)
- Suporte a múltiplos cartões por transação
- Políticas de depósito configuráveis por serviço
- Taxas de cancelamento e no-show automáticas
- Cartão on file para cobranças futuras
- Pagamentos independentes para profissionais autônomos (routing direto)

**Fresha Capital (parceria com Adyen):**
- Empréstimos de $500 a $50.000
- Pré-aprovação automática baseada em histórico de vendas na plataforma
- Liberação em horas ou no mesmo dia útil
- Repagamento como % fixa das vendas diárias (1% a 15%)
- Prazo de até 9 meses
- Sem multa por pagamento antecipado
- Disponível em: EUA, Reino Unido, Austrália, Canadá, Países Baixos, Finlândia, Suécia

**Segurança de Pagamentos:**
- PCI DSS compliant
- Criptografia TLS para transmissão
- Tokenização de dados de cartão
- Firewalls avançados

## Pontos Fortes

1. **Marketplace global com 25M+ usuários**: canal de aquisição de clientes embutido sem custo de marketing para o negócio — diferencial que nenhum concorrente replica na mesma escala.

2. **Modelo de comissão inteligente**: 20% apenas na primeira visita, zero para retorno. Incentiva retenção e elimina fricção para visitas recorrentes.

3. **Integração vertical completa**: do agendamento ao pagamento ao marketing ao capital financeiro — tudo numa plataforma, eliminando necessidade de múltiplas ferramentas.

4. **AI Intelligent Scheduling**: Dynamic Reassignment, precificação dinâmica, gestão de recursos físicos — diferencial técnico significativo que reduz slots vazios.

5. **Fresha Capital**: empréstimos embutidos baseados em histórico de vendas — proposta de valor financeira que aprofunda o lock-in.

6. **Escala global**: 120 países, suporte multi-idioma, presença em mercados emergentes e desenvolvidos.

7. **Freemium como motor de aquisição**: barreira de entrada mínima gera base massiva de negócios parceiros.

8. **AEO (Answer Engine Optimization)**: 68,3% de AI Visibility Score — líder em visibilidade em assistentes de IA como ChatGPT quando salões buscam recomendações.

9. **Integrações nativas com Google, Facebook e Instagram**: gratuitas e em tempo real.

10. **PCI DSS + GDPR**: compliance de nível enterprise.

---

## Pontos Fracos / Gaps

1. **Obrigatoriedade de conta do cliente para agendar**: cria atrito na conversão de clientes, especialmente os menos tech-savvy.

2. **Lock-in no processador de pagamento**: negócios são forçados a usar o Fresha Pay. Sem opção de Stripe, PayPal ou outros processadores.

3. **20% de comissão pode ser alto**: para salões de alto volume com muitos clientes novos, o custo pode superar o de concorrentes com mensalidade fixa.

4. **Personalização limitada**: a booking page tem pouca customização de design; não embute diretamente no site do negócio.

5. **Sem integrações com ERPs, CRMs externos ou plataformas de automação** (Zapier, HubSpot, Salesforce).

6. **Suporte ao cliente inconsistente**: suporte telefônico limitado a planos pagos; usuários em planos menores dependem de chat e email.

7. **Mobile (framework não identificado)**: tecnologia do app não é transparente; pode ser um risco de manutenção a longo prazo.

8. **Relatórios básicos no plano padrão**: analytics avançados exigem add-on pago.

9. **Mudança de promessa "free forever"**: gerou desconfiança e perda de clientes ao introduzir cobranças mensais em 2025.

10. **Sem integração nativa com Quickbooks** (muito usado no mercado americano).

## O que Replicar para um Sistema Similar

### Estratégias de Negócio

| O que replicar | Como adaptar |
|---|---|
| **Marketplace + SaaS integrado** | Canal de descoberta próprio reduz CAC para os negócios parceiros e cria flywheel de crescimento |
| **Comissão apenas no 1º agendamento** | Incentiva adoção rápida sem penalizar negócios com clientes fiéis; melhor que comissão por transação |
| **Freemium com upsell por features** | Plano gratuito ou barato gera base grande; add-ons para features avançadas monetizam os mais comprometidos |
| **Capital embutido baseado em dados** | Serviço financeiro usando histórico de transações aprofunda lock-in e gera receita adicional |
| **AEO + SEO integrado** | Criar conteúdo otimizado para ser citado por IAs além de rankar no Google |

### Funcionalidades Prioritárias para Replicar

| Prioridade | Feature | Por quê |
|---|---|---|
| Alta | Calendário com múltiplos profissionais e drag-and-drop | Core do produto |
| Alta | Agendamento online 24/7 com booking page customizável | Eliminação do telefone |
| Alta | Lembretes automáticos SMS/WhatsApp/email | Redução de no-shows |
| Alta | Lista de espera automática | Preenchimento passivo de agenda |
| Alta | POS integrado ao calendário | Zero fricção no checkout |
| Alta | CRM com histórico, notas, fórmulas | Retenção de conhecimento sobre clientes |
| Alta | Gift cards e pacotes de serviços | Receita antecipada |
| Alta | Formulários de consulta digital | Profissionalismo e compliance |
| Média | Marketing automatizado (aniversário, reengajamento) | Retenção passiva |
| Média | Comissões e payroll de equipe | Gestão de profissionais |
| Média | Relatórios básicos de receita e performance | Visibilidade do negócio |
| Média | Integrações com Google/Instagram/Facebook | Canais de booking adicionais |
| Média | Depósitos e políticas de cancelamento | Proteção contra no-show |
| Baixa | AI Dynamic Reassignment | Complexidade técnica alta, valor relevante só em escala |
| Baixa | Fresha Capital / empréstimos | Requer licença financeira |
| Baixa | Data Connector / BI avançado | Para segmento enterprise |

### Tecnologia Recomendada para Replicar

| Camada | Stack sugerida com base no Fresha |
|---|---|
| Frontend | Next.js + TypeScript + React |
| Backend API | Node.js (NestJS) ou Ruby on Rails + GraphQL |
| Microserviços | Elixir/Phoenix para alta concorrência (mensagens, notificações) |
| Banco de dados | PostgreSQL (principal) + Redis (cache/filas) |
| Cloud | AWS (EC2, RDS, S3, CloudFront) |
| Containers | Docker + Kubernetes |
| Pagamentos | Stripe (mais flexível que Adyen para startups) |
| Notificações | Twilio (SMS) + SendGrid (email) + Meta Cloud API (WhatsApp) |
| Analytics | Metabase ou Redash sobre PostgreSQL |
| CI/CD | GitHub Actions + ArgoCD |
| Observabilidade | Datadog ou Grafana + Prometheus |
| Mobile | React Native (reuso de código web) |

### Gaps do Fresha = Oportunidades de Diferenciação

1. **Permitir agendamento sem criar conta** (guest checkout) — elimina maior fricção do Fresha.
2. **Processador de pagamento flexível** — permitir Stripe, PagSeguro, Mercado Pago, etc.
3. **Booking page embutível no website** via iframe ou widget nativo.
4. **Integrações com ferramentas populares** (Zapier, Google Sheets, Quickbooks, Conta Azul).
5. **Política de cancelamento com tiers** (ex: 25% cancelado com 24h, 50% com menos de 12h).
6. **Transparência de preços** — não mudar de "gratuito para sempre" abruptamente.
7. **Suporte em português com atendimento humano** para o mercado brasileiro.
